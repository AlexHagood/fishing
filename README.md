# GLOP 3 (FISHING EDITION)
The newest version of Glop is the fishiest yet! Explore the world, catch fish, and manage your inventory!

# ONBOARDING (WINDOWS)
1. Open Powershell (Windows+X, I)
2. Install Git `winget install --id Git.Git -e --source winget`
3. Install GitHub desktop if you want `winget install -e --id GitHub.GitHubDesktop`
4. Install .NET (THE SECOND GRAY BUTTON) Godot from https://godotengine.org/download/windows/
5. Download the source code `git clone https://github.com/AlexHagood/fishing.git`
6. Open Godot Mono, Hit Import at the top left. Find where you downloaded the source code, and select the project.godot
7. Once launched, press F5 to test run the game. Success!
